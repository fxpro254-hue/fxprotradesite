Right-click element → \"Inspect\"\n3. In DevTools, look for theme toggle\n4. Alternative: `about:config` → \"ui.systemUsesDarkTheme\" (1=dark, 0=light)\n\n### Safari\n1. System Preferences → General\n2. Appearance: Light/Dark/Auto\n3. Refresh page to see change\n\n---\n\n## Common Use Cases\n\n### Background Color\n```scss\nbackground: #F2F3F4;  // Light\n// or\n@media (prefers-color-scheme: dark) {\n    background: #151717;  // Dark\n}\n```\n\n### Text Color\n```scss\ncolor: #333333;
